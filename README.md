## ðŸ“– Overview

CoursesOnline is a full-stack application designed to manage online courses. It features a robust RESTful API built with ASP.NET Core following Clean Architecture principles, ensuring scalability and maintainability. The frontend is a modern Single Page Application (SPA) built with React and Vite, offering a responsive and interactive user experience.

## ðŸ“‚ Project Structure

The solution follows a Clean Architecture approach:

- **`ApiRest`**: The entry point of the application. Contains Controllers, Configuration, and API setup.
- **`Application`**: Contains business logic, use cases, DTOs, and interfaces. It depends only on the Domain.
- **`Domain`**: The core of the application. Contains Entities, Value Objects, and Enterprise Logic. No external dependencies.
- **`Infrastructure`**: Implementation of interfaces defined in Application (Data Access, External Services). Contains the `ApplicationDbContext`.
- **`CursosOnlineFront`**: The React frontend application source code.

## ðŸš€ Running the Project

### Option 1: Docker (Recommended)

The easiest way to run the entire stack is using Docker Compose.

1.  **Navigate to the root directory**:
    ```bash
    cd /path/to/CoursesOnline
    ```

2.  **Build and Start Containers**:
    ```bash
    docker-compose up --build
    ```

3.  **Access the Application**:
    - **Frontend**: [http://localhost:3001](http://localhost:3001)
    - **Backend API**: [http://localhost:5282](http://localhost:5282) (Swagger: [http://localhost:5282/swagger](http://localhost:5282/swagger))

### Option 2: Local Development

If you prefer to run services individually:

**Backend:**
1.  Navigate to the API directory:
    ```bash
    cd ApiRest
    ```
2.  Run the application:
    ```bash
    dotnet run
    ```
    The API will start on `http://localhost:5282` (or the port defined in `launchSettings.json`).

**Frontend:**
1.  Navigate to the Frontend directory:
    ```bash
    cd CursosOnlineFront
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
    The frontend will typically start on `http://localhost:5173` (default Vite port) or `http://localhost:3001` if configured.
